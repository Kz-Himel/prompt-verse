# 📖 Project Overview

PromptVesre is a full-stack AI Prompt Sharing & Marketplace Platform where users can discover, publish, review, bookmark, and purchase premium AI prompts.

The platform supports multiple AI tools including ChatGPT, Gemini, Claude, Midjourney, Copilot, and many others.

Users can browse trending prompts, search with advanced filtering, review prompt quality, bookmark favorites, copy prompt content, and purchase Premium access through Stripe Payment Integration.

The system also includes complete Role-Based Authentication, JWT Authorization, Better Auth Authentication, Analytics Dashboard, Prompt Moderation, Reporting System, and Admin Management Panel.
